Compute the manipulability Yoshikawa index

Parameters:
  • robot (orpy.Robot) – OpenRAVE robot object
  • link_name (str) – The name of link for which the index will be computed
  • translation_only (boolean, optional) – If True use only the translation part of the Jacobian matrix to compute the index. If False use the translation and rotation.
  • penalize_jnt_limits (boolean, optional) – If True penalize/attenuate the manipulability proportionally to the distance to the joint limits of the robot
Returns:

index – The Yoshikawa manipulability index
Return type:

float
raveutils.kinematics.find_ik_solutions(robot, target, iktype, collision_check=True, freeinc=0.087)[source]¶

Find all the possible IK solutions

Parameters:
  • robot (orpy.Robot) – OpenRAVE robot object
  • target (array_like or orpy.Ray) – The target in Cartesian space. A target can be defined as a Ray (position and direction) which is 5D or an homogeneous transformation which is 6D
  • iktype (orpy.IkParameterizationType) –

    Inverse kinematics type to use. Supported values:

    • orpy.IkParameterizationType.Transform6D
    • orpy.IkParameterizationType.TranslationDirection5D
  • collision_check (bool, optional) – If true find only collision-free solutions
Returns:

solutions – The found IK solutions
Return type:

list
